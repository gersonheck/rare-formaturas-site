# Rare Formaturas — Site

Site institucional da Rare Formaturas, desenvolvido com Next.js 14, Tailwind CSS e Framer Motion.

## Requisitos

- Node.js 18+
- npm 9+

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) no navegador.

## Build de produção

```bash
npm run build
npm run start
```
